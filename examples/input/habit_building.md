# 如何养成好习惯

## 第一步：选择小习惯
从不到2分钟的小事开始。比如：
- 读一页书
- 做一个俯卧撑
- 写一句话

## 第二步：绑定现有流程
把它和每天已经在做的事情关联。比如：
- 早上刷牙后 → 冥想1分钟
- 午饭后 → 走楼梯
- 晚上睡前 → 读一页书

## 第三步：让它显而易见
创建视觉提示和提醒：
- 把运动服放在床头
- 把书放在枕头上
- 设置手机提醒

## 第四步：让它有吸引力
把想做的事和需要做的事绑定：
- 只有运动时才听喜欢的播客
- 只有学习时才喝喜欢的咖啡

## 第五步：追踪进度
使用习惯追踪器：
- 打卡日历
- 习惯追踪APP
- 简单的笔记本记录
